# Employee Management System

This project demonstrates an employee management system using SQL.

## Features:
- Employee and department management
- Salary analysis
- Attendance tracking

## Database Tables:
- Employees
- Departments
- Attendance

## SQL Concepts Used:
- Joins
- Group By & Aggregations
- Filtering

## Insights:
- Department-wise salary analysis
- Highest paid employee
- Attendance reports

## Tools Used:
- MySQL
- SQL
